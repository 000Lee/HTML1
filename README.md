# VS Code 단축키

## 이동 관련
- **Home**  
  행의 맨앞으로 이동  

- **End**  
  행의 맨뒤로 이동  

- **Ctrl + Enter**  
  다음 라인으로 이동  

## 행 이동 및 복사
- **Alt + ↑ / ↓**  
  커서가 위치한 행, 선택한 행을 위 혹은 아래로 이동  
  (여러 행 대응 가능)

- **Alt + Shift + ↑ / ↓**  
  커서가 위치한 행, 선택한 행을 복사하며 이동  
  (여러 행 대응 가능)

## 복사 / 붙여넣기 / 잘라내기
- **Ctrl + C, Ctrl + V, Ctrl + X**  
  선택 범위를 복사, 붙여넣기, 잘라내기  
  블록 선택하지 않은 상태에서 사용 가능

## 들여쓰기
- **Ctrl + ] , Ctrl + [**  
  오른쪽 들여쓰기, 왼쪽 들여쓰기

## 멀티 커서 편집
- **Ctrl + Shift + L**  
  같은 글자 한꺼번에 선택, 수정 가능

## 주석
- **Ctrl + /**  
  코드에 대한 설명을 할 때 주로 쓰임  
  프로그램에 영향을 미치지 않음

## 탐색기
- **Ctrl + B**  
  폴더 목록 접었다 펴기

## 터미널
- **Ctrl + ` (Backtick)**  
  문제, 출력, 디버그 콘솔, 터미널, 포트 창 표시
